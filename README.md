# dp-migrering

Backend-applikasjon som håndterer innsyn i Arena saker og etterhvert migrering av saker fra `Arena` til `dp-sak`.

## Komme i gang

Gradle brukes som byggverktøy og er bundlet inn.

`./gradlew build`
